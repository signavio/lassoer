
# sqltypes

<!-- badges: start -->
<!-- badges: end -->

Write the schema of a csv file in standard SQL datatypes with dates formatted according to the golang time format.

## Installation

You can install the released version of sqltypes from [CRAN](https://CRAN.R-project.org) with:

``` r
install.packages("sqltypes")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(sqltypes)
## basic example code
```
